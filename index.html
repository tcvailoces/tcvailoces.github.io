Polaroid Prints

I developed the "Polaroid Prints" project using Vanilla JavaScript to simplify the bulk creation and printing of polaroid-style designs, tailored for my personal use. Initially, I faced challenges manually placing photos in a Photoshop template, especially with the increasing number of orders. Downloading client images from Facebook added complexity.

To streamline my workflow, I designed a straightforward form for inputting image links. Clicking the "Print" button now triggers Chrome's built-in print feature, eliminating manual adjustments and significantly reducing processing time. This efficient solution relies on HTML, CSS, and JavaScript ES6.

Looking ahead, I envision implementing additional features:

  1. 1. Drag and Drop: Enhance user experience by allowing easy drag-and-drop functionality.
  2. 2. Fetch from Facebook and Google Drive: Streamline image input by incorporating Facebook album and Google Drive folder retrieval through API integration.
  3. 3. Image Filters: Enable users to set filters on each image, adding creative customization to the Polaroid prints.

This personalized tool aims to continually enhance efficiency and simplicity for my personal bulk printing needs.

Piso-Wifi Voucher Template Generator

For my personal project, the "Piso-Wifi Voucher Template Generator," I utilized HTML, CSS and Vanilla JavaScript to enhance the design of existing voucher tickets for Piso-Wifi machines. Unlike the standard plain text vouchers, my version incorporates a strategic marketing approach by showcasing a list of additional services I offer in the local area.

For context, a Piso-Wifi is a coin or voucher-operated wifi machine that provides internet access for users who want to connect. The challenge with the usual design was its simplicity, displaying only the voucher code on the paper.

To address this, I created a form capable of accepting hundreds of voucher codes simultaneously. Through JavaScript, the project dynamically loops through each instance, integrating the codes into the redesigned template. This innovative solution not only enhances the voucher's visual appeal but also serves as a marketing tool by prominently featuring the range of services available in the area.

This site is crafted with HTML, CSS (Tailwind) and JavaScript, a reflection of simplicity and style.
